# Jal-Sanchay Tracker 💧

A modern Android application developed to monitor rainwater harvesting, track water conservation, visualize tank storage, and encourage sustainable water management practices using real-time analytics and smart insights.

---

## 🚀 Features

- 🔐 User Login and Authentication
- 🌧️ Rainfall Data Entry and Tracking
- 💧 Animated Water Tank Visualization
- 📊 Water Harvesting Analytics and Charts
- 📄 Monthly PDF Report Generation
- 📈 Historical Rainfall and Water Saving Records
- 🔔 Smart Notifications and Alerts
- 🌱 Environmental Impact Monitoring
- 🏠 Estimated Household Water Usage Calculation
- ☁️ Firebase Firestore Cloud Storage
- 🎨 Modern Jetpack Compose UI Design
- ⚡ Fast and Responsive Performance

---

## 🛠️ Technologies Used

- Kotlin
- Jetpack Compose
- Firebase Authentication
- Firebase Firestore
- MPAndroidChart
- Material Design 3
- Android Studio
