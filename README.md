Jal-Sanchay Tracker 💧
An Android application developed to monitor rainwater harvesting, track water conservation, and promote sustainable water management practices. The app helps users calculate harvested rainwater, visualize tank storage, generate analytics reports, and understand environmental impact through a simple and modern interface.

🚀 Features
* 🔐 User Login and Authentication
* 🌧️ Rainfall Data Entry and Tracking
* 💧 Animated Water Tank Visualization
* 📊 Water Harvesting Analytics and Charts
* 📄 Monthly PDF Report Generation
* 📈 Historical Rainfall and Water Saving Records
* 🔔 Smart Notifications and Alerts
* 🌱 Environmental Impact Monitoring
* 🏠 Estimated Household Water Usage Calculation
* ☁️ Firebase Firestore Cloud Storage
* 🎨 Modern Jetpack Compose UI Design
* ⚡ Fast and Responsive Performance

🛠️ Technologies Used
* Kotlin
* Jetpack Compose
* Firebase Authentication
* Firebase Firestore
* MPAndroidChart
* Material Design 3
* Android Studio
  
📂 Project Structure
com.kokila.jalsanchay

ui/ → Jetpack Compose UI screens and components
screens/ → Dashboard, History, Weather, Tips, Login, and Settings screens
firebase/ → Firebase Authentication and Firestore integration
charts/ → Analytics graph and chart handling
reports/ → PDF report generation utilities
notifications/ → Notification and smart alert management
utils/ → Helper classes, constants, and common utilities
theme/ → App themes, colors, and typography
res/ → Drawables, icons, strings, and app resources
 
 ⚙️ Setup Instructions
 Clone Repository
 git clone https://github.com/kokila8792/MM-Jal-Sanchay-Tracker.git

Open in Android Studio
1. Open Android Studio
2. Click Open Project
3. Select the project folder
4. Wait for Gradle Sync
5. Run the application

📱 Main Modules
Dashboard Screen
Displays water tank animation, total savings, smart alerts, and analytics.

Weather Screen
Shows rainfall and weather-related information.

History Screen
Displays historical rainwater harvesting records and analytics charts.

Tips Screen
Provides water conservation and rainwater harvesting tips.

Settings Screen
Allows user configuration and preferences management.

PDF Report Generator
Generates monthly rainwater harvesting reports.
