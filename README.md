# Jal-Sanchay Tracker 💧

A modern Android application developed to monitor rainwater harvesting, track water conservation, visualize tank storage, and encourage sustainable water management practices using real-time analytics and smart insights.

---

## 🚀 Features

- 🔐 User Login and Authentication
- 🌧️ Rainfall Data Entry and Tracking
- 💧 Animated Water Tank Visualization
- 📊 Water Harvesting Analytics and Charts
- 📄 Monthly PDF Report Generation
- 📈 Historical Rainfall and Water Saving Records
- 🔔 Smart Notifications and Alerts
- 🌱 Environmental Impact Monitoring
- 🏠 Estimated Household Water Usage Calculation
- ☁️ Firebase Firestore Cloud Storage
- 🎨 Modern Jetpack Compose UI Design
- ⚡ Fast and Responsive Performance

---

## 🛠️ Technologies Used

- Kotlin
- Jetpack Compose
- Firebase Authentication
- Firebase Firestore
- MPAndroidChart
- Material Design 3
- Android Studio

### reports/

Notification and smart alert management

---

### theme/

## ⚙️ Setup Instructions

### Clone Repository

```bash
git clone https://github.com/kokila8792/MM-Jal-Sanchay-Tracker.git
```

---

### Open in Android Studio

1. Open Android Studio  
2. Click Open Project  
3. Select the project folder  
4. Wait for Gradle Sync  
5. Run the application

## 📱 Main Modules

### Dashboard Screen
Displays animated water tank levels, total water savings, analytics, and smart recommendations.

---

### Login Screen
Handles user authentication and secure login.

---

### History Screen
Displays historical rainfall records and harvesting analytics charts.

---

### Weather Screen
Shows rainfall and weather-related information for harvesting awareness.

---

### Tips Screen
Provides rainwater harvesting and water conservation tips.

---

### Settings Screen
Allows user configuration and preferences management.

---

### PDF Report Generator
Generates detailed monthly rainwater harvesting reports.
